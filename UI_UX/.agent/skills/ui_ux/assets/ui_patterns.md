# Common UI Patterns

Reference library of proven UI patterns.

---

## Navigation Patterns

### Tab Bar (Mobile)
- 3-5 items max
- Icons + labels
- Highlight active state
- Fixed at bottom

### Sidebar (Desktop)
- Collapsible
- Icons + text
- Nested items with indentation
- Highlight active section

### Breadcrumbs
- Show hierarchy
- Clickable links except current page
- Separator: ` > ` or ` / `

---

## Form Patterns

### Inline Validation
- Validate on blur
- Show error immediately below field
- Green check for valid
- Red text for errors

### Multi-Step Form
- Progress indicator
- Save progress between steps
- "Back" and "Next" buttons
- Review before submit

### Search
- Prominent placement
- Placeholder text
- Clear button when filled
- Autocomplete/suggestions

---

## Feedback Patterns

### Toast/Snackbar
- Brief, auto-dismiss
- Action button (Undo)
- Bottom of screen
- 3-5 seconds duration

### Modal/Dialog
- Overlay background
- Clear close button
- Focus trap inside
- Esc to close

### Loading States
- Skeleton screens > spinners
- Progress bar for known duration
- Loading text for context

---

## Action Patterns

### Primary/Secondary Buttons
- Primary: Filled, prominent color
- Secondary: Outline or subtle
- One primary CTA per view

### Destructive Actions
- Red color
- Confirmation dialog
- Undo option when possible

### Disabled States
- Reduced opacity
- Explain why disabled (tooltip)
- Don't hide, show disabled

---

## List Patterns

### Infinite Scroll
- Loading indicator at bottom
- "Back to top" button
- Remember scroll position

### Pagination
- Page numbers
- Prev/Next buttons
- "Page X of Y" indicator

### Filters
- Sidebar or dropdown
- Clear filters button
- Show active filter count
