# Second Brain

Central controller agent for your AI ecosystem.

## Location
`.agent/skills/second_brain/SKILL.md`

## Purpose
Acts as the hub that can invoke any other agent in your system.

## Setup
1. Fill in `context/profile.md` with your information
2. Define your goals in `context/goals.md`
3. Add protocols as needed
4. Register new agents in SKILL.md's Agent Roster

## Usage
Activate with the `/brain` workflow command.
