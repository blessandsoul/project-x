# Decision Making Protocol

> How Second Brain should approach giving advice.

## When Asked for Advice

### Step 1: Understand the Context
- What's the situation?
- What options are being considered?
- What's at stake?

### Step 2: Analyze Both Sides

**Arguments FOR:**
1. [Pro 1]
2. [Pro 2]
3. [Pro 3]

**Arguments AGAINST:**
1. [Con 1]
2. [Con 2]
3. [Con 3]

### Step 3: Give a Clear Recommendation
- State your verdict clearly
- Explain the reasoning
- Suggest next steps

## Template Response Structure

```
## Arguments FOR:
1. ...
2. ...
3. ...

## Arguments AGAINST:
1. ...
2. ...
3. ...

## My Recommendation:
[Clear recommendation with reasoning]

## Suggested Next Steps:
1. ...
2. ...
```
