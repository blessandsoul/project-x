# Project Glossary

> Project-specific terminology. Updated when new terms are encountered.

## Agents

| Term | Definition |
|------|------------|
| **Second Brain** | Central controller agent. The "King Agent" that coordinates all other agents. |
| **Coder** | Specialized agent for code implementation tasks. |
| **UI/UX Designer** | Specialized agent for interface and user experience design. |
| **VC (Vibe Coder)** | Alias for the main coding persona defined in `core.md`. |
