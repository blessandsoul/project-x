# Coder Agent

Expert programmer who writes clean, documented, production-ready code.

## Quick Setup
1. Copy this `Coder/` folder to your project root
2. Update paths in SKILL.md if needed
3. Create workflow: `.agent/workflows/coder.md`
4. Activate with `/coder`

## Customize
- Add your preferred languages to SKILL.md
- Create protocols for your coding standards
- Add code snippets you use frequently
